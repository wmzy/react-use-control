# React Use Control

> Make React component state controllable — seamless controlled/uncontrolled support via a lightweight control object.

## What is a Control?

A **control** is a lightweight object that carries state through the component tree. When you call `useControl`, you get back `[value, setValue, control]` — the control can be passed as a prop to child components, letting them **share the same state** without Context or state lifting boilerplate.

## Core API

| API | Description |
|-----|-------------|
| `useControl(control?, initial)` | Main hook — returns `[value, setValue, control]` |
| `useThru(control, interceptor)` | Wrap a control with a state/setter transform |
| `controlEqual(prev, next)` | Custom comparator for `React.memo` |
| `isControl(x)` | Type guard for control objects |
| `mapState(fn)` | Transform the value children see |
| `mapSetter(fn)` | Transform values before they reach setState |
| `watch(onChange)` | Side-effect on state changes |

## Concepts

### Uncontrolled Mode

When no control is passed, the component manages its own state:

```jsx
function Counter() {
  const [count, setCount] = useControl(null, 0);
  return <button onClick={() => setCount(c => c + 1)}>{count}</button>;
}
```

### Controlled Mode

Pass a control from a parent to share state:

```jsx
function Parent() {
  const [count, setCount, control] = useControl(null, 0);
  return (
    <div>
      <Counter count={control} />
      <button onClick={() => setCount(0)}>Reset</button>
    </div>
  );
}
```

### Sibling Sharing

Pass the same control to multiple children — they all share the same state:

```jsx
function App() {
  const [, , control] = useControl(null, 0);
  return (
    <div>
      <Counter count={control} />
      <Counter count={control} />
    </div>
  );
}
```

### State Transforms

Use `useThru` to intercept and transform state as it flows through the tree:

```jsx
import { useThru, mapSetter } from 'react-use-control';

function DoubleCounter({ count }) {
  const control = useThru(count, mapSetter(v => v * 2));
  return <Counter count={control} />;
}
```
