# React Use Control

> Make React component state controllable — seamless controlled/uncontrolled support via a lightweight control object.

## What is a Control?

A **control** is a lightweight object that carries state through the component tree. When you call `useControl`, you get back `[value, setValue, control]` — the control can be passed as a prop to child components, letting them **share the same state** without Context or state lifting boilerplate.

## Concepts

### Uncontrolled Mode

When no control is passed, the component manages its own state:

```jsx
function Counter() {
  const [count, setCount] = useControl(null, 0);
  return <button onClick={() => setCount(c => c + 1)}>{count}</button>;
}
```

### Controlled Mode

Pass a control from a parent to share state:

```jsx
function Parent() {
  const [count, setCount, control] = useControl(null, 0);
  return (
    <div style={{ padding: 12, border: '2px solid #4a90d9', borderRadius: 8 }}>
      <div style={{ fontSize: 12, color: '#4a90d9', marginBottom: 8 }}>Parent</div>
      <div style={{ padding: 8, border: '1px dashed #999', borderRadius: 4 }}>
        <div style={{ fontSize: 12, color: '#999', marginBottom: 4 }}>Child</div>
        <Counter count={control} />
      </div>
      <button onClick={() => setCount(0)}>Reset</button>
    </div>
  );
}
```

### Sibling Sharing

Pass the same control to multiple children — they all share the same state:

```jsx
function App() {
  const [, , control] = useControl(null, 0);
  return (
    <div style={{ padding: 12, border: '2px solid #4a90d9', borderRadius: 8 }}>
      <div style={{ fontSize: 12, color: '#4a90d9', marginBottom: 8 }}>Parent</div>
      <div style={{ display: 'flex', gap: 12 }}>
        <div style={{ padding: 8, border: '1px dashed #999', borderRadius: 4 }}>
          <div style={{ fontSize: 12, color: '#999', marginBottom: 4 }}>Child A</div>
          <Counter count={control} />
        </div>
        <div style={{ padding: 8, border: '1px dashed #999', borderRadius: 4 }}>
          <div style={{ fontSize: 12, color: '#999', marginBottom: 4 }}>Child B</div>
          <Counter count={control} />
        </div>
      </div>
    </div>
  );
}
```

### State Transforms

Use `useThru` to intercept and transform state as it flows through the tree:

```jsx
import { useThru, mapSetter } from 'react-use-control';

function DoubleCounter({ count }) {
  const control = useThru(count, mapSetter(v => v * 2));
  return <Counter count={control} />;
}
```

### Cross-Layer Sharing

Pass a control through multiple layers of wrapper components — no Context, no callback drilling:

```jsx
function Input({ temperature }) {
  const [temp, setTemp] = useControl(temperature, 20);
  return <button onClick={() => setTemp(t => t + 1)}>{temp}°C</button>;
}

function Card({ temperature }) {
  const [temp] = useControl(temperature, 20);
  return (
    <div style={{ padding: 12, border: '2px solid #4a90d9', borderRadius: 8 }}>
      <div style={{ fontSize: 12, color: '#4a90d9', marginBottom: 8 }}>Card</div>
      <div>{temp}°C</div>
      <div style={{ padding: 8, border: '1px dashed #999', borderRadius: 4 }}>
        <div style={{ fontSize: 12, color: '#999', marginBottom: 4 }}>Input</div>
        <Input temperature={temperature} />
      </div>
    </div>
  );
}

function App() {
  const [temp, setTemp, control] = useControl(null, 20);
  return (
    <div style={{ padding: 12, border: '2px solid #4a90d9', borderRadius: 8 }}>
      <div style={{ fontSize: 12, color: '#4a90d9', marginBottom: 8 }}>App</div>
      <div>Top-level: {temp}°C</div>
      <Card temperature={control} />
      <button onClick={() => setTemp(20)}>Reset</button>
    </div>
  );
}
```
